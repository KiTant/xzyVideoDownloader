# xzyVideoDownloader
Application that allows you to easily download videos from the internet. Simply paste a video link, and the program will save it to your device.

## Used modules:
- customtkinter
- yt_dlp
- CTkMessagebox
- CTkMenuBar
- CTkListbox
- queue
- os
- win11toast
- json
- requests
- time
- threading
